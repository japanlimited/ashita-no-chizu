# あしたの地図 MVP

要件定義に沿って作成した、ブラウザで動作するWebアプリMVPです。

## 動かし方

通常のブラウザでは `index.html` を直接開けます。

Codexのアプリ内ブラウザでは `file://` がブロックされる場合があるため、次のどちらかでローカルサーバーを起動してください。

```powershell
cd C:\Users\japan\OneDrive\Desktop\環境構築用\ashita-no-chizu
node server.js
```

または `start.bat` を実行します。

起動後、次のURLを開きます。

```text
http://127.0.0.1:4173/index.html
```

- 診断結果画面
- 道のり型未来マップ
- 目標提案と調整
- 明日からの一歩
- 9:16の未来応援図
- ホーム画面
- マイページ
- データ削除

## AI連携について

MVPとして、現時点ではブラウザ単体で動くローカル分析フォールバックを実装しています。
実際のOpenAI API連携を入れる場合は、ブラウザにAPIキーを置かず、Next.js API Routeやサーバー側のエンドポイントから `buildAiPrompt()` 相当の内容を送信する構成にしてください。
